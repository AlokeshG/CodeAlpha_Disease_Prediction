# Diabetes Prediction System

## Project Overview
This project predicts the possibility of diabetes using Machine Learning algorithms.

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML/CSS

## Features
- Predicts diabetes risk
- Displays prediction probability
- Web-based interface

## Algorithms Used
- Random Forest Classifier

## Dataset
Pima Indians Diabetes Dataset

## Accuracy
Approx. 90% Accuracy
