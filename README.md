# ❤️ Heart Disease Prediction System

An AI-powered Heart Disease Prediction System developed using Machine Learning and Flask that predicts the likelihood of heart disease based on patient medical parameters.

The project uses the Heart Disease dataset (`heart_statlog_cleveland_hungary_final.csv`) and applies the Random Forest Classifier to analyze patient health data and classify heart disease risk levels into Low, Medium, and High categories.

## 🚀 Features

- Heart disease risk prediction using Machine Learning
- Random Forest Classification model
- Risk level classification:
  - ✅ Low Risk
  - ⚠ Medium Risk
  - 🚨 High Risk
- Probability score generation
- Modern responsive healthcare UI
- Flask-based web application
- Real-time prediction system

## 🩺 Input Parameters

The system predicts risk using:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise Angina
- Old Peak
- ST Slope
